## Back-end для приложения для волонтеров

# Запуск приложения

1. Поменять адресс сервера в *application.conf*
2. Запустить "*./gradlew build*"

# Используемые технологии

- Язык программирования: Kotlin
- Фреймворки: Ktor, Exposed
